# Sonar v0.2.1

Release date: 2026-05-17

## Summary

Sonar v0.2.1 is a patch release that aligns the MCP server surface with the
`0.2.0` runtime behavior.

This release adds:

- MCP tool descriptions that call out PDF extraction and semantic topic pruning
- MCP wrapper tests covering PDF extraction passthrough
- MCP wrapper tests covering topic-collection semantic filtering passthrough

## Highlights

### MCP descriptions now match runtime semantics

- `sonar_extract` now advertises PDF-capable extraction
- `sonar_collect_sources_for_topic` now advertises semantic relevance pruning
- server instructions now mention both PDF extraction and embeddings-backed
  topic filtering

### MCP regression coverage expanded

- added tests proving the MCP wrapper forwards PDF extraction responses
- added tests proving the MCP wrapper forwards topic-collection filtering
  responses and warnings
